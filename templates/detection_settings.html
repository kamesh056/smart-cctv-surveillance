How Detection Range Works
The face recognizer resizes each frame before running detection.

scale_factor: A value of 0.25 means faces need to be very close and large to be detected. Setting it to 0.75–1.0 preserves more image detail, allowing detection of faces that are further away from the camera.

upsample: Internally upscales the image so the detector can find smaller faces. Setting to 2 significantly increases range but doubles processing time.

After changing settings, restart the CCTV Engine from the Dashboard for changes to take effect.
